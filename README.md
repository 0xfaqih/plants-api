# plants-api

## Petunjuk Instalasi

1. **Clone Repositori:**
   ```bash
   git clone https://github.com/0xfaqih/plants-api.git
2. **Install Dependensi:**
   ```bash
   npm install
3. **Buat File .env,**
   ex:
   ```
   PORT=3000
   HOST=localhost
   MONGODB_URI=mongodb://localhost:27017/nama_database
2. **Jalankan Aplikasi**
   ```bash
   npm run start
